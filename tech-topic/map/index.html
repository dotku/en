International Map Service List

I'm working on a map related project recently. I have working on this project over a month. Recently, I'm getting stacked on choosing a proper map service. The map service should have international language support.

At first, I was assumed Google map would be the best match, but later I found that there is few data entries just don't work properly, for example, when I looking for "台北" in simplified Chinese characters it would returns "新北市." Well, in English they are pretty close matched, "台北" is Taibei in English and "新北市" is "New Taibei" in English. That just only one sample. There are many other Asian locations are mistakenly matched. Since the geo location is very important part in the project, so I star looking for a better service, and finally, Here API fits our needs, and the complete list of map service I have gone through are list below. Some folks might want check them out.
